INSPIRE DATASET METADATA TEMPLATE

Dit repository bevat:

  • een template voor het eenvoudig aanmaken van een geldig INSPIRE dataset metadata record in nationaalgeoregister.nl (NGR)
  • een handleiding voor het invullen van dit template in de metadata editor van het NGR

Dit template is aangemaakt aan de hand van het Nederlands metadata profiel op ISO 19115 voor geografie versie 2.1.0.

Handleiding

Deze handleiding beschrijft hoe een valide INSPIRE dataset metadata record aan te maken op basis van het door PDOK aangemaakte Template Nederlands metadata profiel op ISO 19115 voor geografie 2.1.0 - INSPIRE.

1. Aanmaken nieuw metadata record op basis van template

  • Log in op het nationaalgeoregister.nl en kies via het menu Metadata beheer > Record toevoegen
  • Kies dan voor Dataset als type document
  • En Template Nederlands metadata profiel op ISO 19115 voor geografie 2.1.0 - INSPIRE als template
  • Kies de juiste groep voor het nieuwe metadata record
  • En klik op Aanmaken

Na het aanmaken van het nieuwe metadata record wordt het nieuwe record geopened in de metadata editor van het NGR.

2. Invullen velden nieuw metadata record

Deze invul instructie gaat er vanuit dat de Nederlands profiel view geselecteerd is in de metadata editor:

2.1 Downloads, views en links

  1. Kleine afbeelding - Vervang de placeholder voorbeeld weergave afbeelding met een van de dataset die beschreven wordt in dit metadata record. Aangeraden wordt vanuit PDOK om een afbeelding te kiezen met afmetingen van 500x500 pixels.
  2. Hyperlinks - Vervang beide links naar de service accespoints, dus voor zowel de dowload als de view service. In het template zijn een WMS en INSPIRE ATOM placeholder service url voor ingevuld, maar hier kunnen ook andere service protocollen gebruikt worden. Vul de volgende velden voor de hyperlink als volgt in:
    • Protocol - Kies juiste waarde uit de lijst
    • URL - Service URL dat linkt naar een capabilities document (of INSPIRE Atom service feed)
    • ❌ https://geodata.nationaalgeoregister.nl/rdinfo/wms
    • ✅ https://geodata.nationaalgeoregister.nl/rdinfo/wms?service=WMS&request=GetCapabilities
    • Naam - De titel van de service
    • Omschrijving - Vaste waarde accessPoint
    • Functie - vaste waarde information
    • Applicatie profiel - Afhankelijk van het type service, zie SpatialDataServiceType codelijst
